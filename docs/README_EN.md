## ğŸš€ Quick Start

### Installation
No installation needed! Just launch the `app.html` file and open it in your browser.

```bash
# Download the file
wget https://github.com/totos30/tournament/archive/refs/heads/main.zip

# Open in your browser
open app.html  # macOS
start app.html # Windows
xdg-open app.html # Linux
```

### Basic Usage

1. **Create a tournament**
   - Select the number of participants
   - Enter names (and optional photos)
   - Click "Continue"

2. **Manage byes** (if applicable)
   - Select seeded players manually
   - Or click "Random Selection"
   - Start the tournament

3. **Play matches**
   - Click on the winner's name for each match
   - Focus automatically follows the next match
   - Use Z/Escape to navigate

4. **Save**
   - Click "Export" at any time
   - The JSON file contains the entire tournament state
